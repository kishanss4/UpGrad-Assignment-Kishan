# Linear Regression From Scratch

This repository contains the submission for the DSMLAI Content Intern assignment.

## Contents
- `Linear_Regression_From_Scratch.ipynb`  
  Jupyter Notebook implementing Linear Regression from scratch without using
  sklearn or statsmodels.

- `Learning_Document_Linear_Regression.pdf`  
  Learning document explaining the notebook as a video-based learning path,
  along with assessment questions.

## Dataset
Housing Prices Dataset (Kaggle)

## Notes
- Only NumPy and Pandas are used
- No high-level machine learning libraries are used
